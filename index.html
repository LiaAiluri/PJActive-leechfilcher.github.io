My Private Projects

Pamaxie - Project Lead

Pamaxie is a publicly accessible content filter aimed at software and hardware developers. Utilizing it, they can determine what type of data was sent through their system and if it is safe for their end users. We plan on developing on an open source, self-hostable API that allows its users to scan media send to it. We plan on detecting malicious files, hate speech, false information and adult content through it. This should hopefully make it easier for website and application developers to focus on their main business instead of developing their own content detection systems. We hope this saves developer time, effort, and money when implementing systems of the same scale as ours.

Grass Shader - Unreal Engine

I started developing this small grass shader just for fun a few years ago. I'm currently working on re-implementing it in GLSL directly. This utilizes a particle system which essentially casts a line onto a texture (The Yellow square in the video below shows it). This texture is then fed into a shader to generate the local map for drawing grass inteaction with the player model or other models too if they have the particle system attached to them. On top of that I implemented a Gerstner wave function to generate the waves in the grass to make it look good / like it's waving in the wind.
