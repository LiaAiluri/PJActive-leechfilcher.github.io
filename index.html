My Private Projects

Pamaxie - Project Lead

Pamaxie is a publicly accessible content filter aimed at software and hardware developers. Utilizing it, they can determine what type of data was sent through their system and if it is safe for their end users. We plan on developing on an open source, self-hostable API that allows its users to scan media send to it. We plan on detecting malicious files, hate speech, false information and adult content through it. This should hopefully make it easier for website and application developers to focus on their main business instead of developing their own content detection systems. We hope this saves developer time, effort and money implementing systems of the same scale as ours.
